# Шрифты TSX-движка (бандл, не ЯД)

Дом брендовых шрифтов для титров/типографики TSX-шаблонов. Лежат **в репо** (а не на ЯД):
раннер получает их вместе с `actions/checkout` → рендер детерминирован на любой машине.
Через ЯД тянуть НЕ надо — это крошечные статичные ассеты, сеть при рендере = лишняя точка отказа.

Загрузка — `remotion/src/fonts.ts` (FontFace + delayRender, ядро remotion).

| Файл | Роль | Где задаётся |
|---|---|---|
| `Lora.ttf` | `TITLE_FONT` — чистый serif для названий/титров | `fonts.ts` → шаблоны |
| `Caveat.ttf` | `HAND_FONT` — рукописный бренд-акцент (тот же, что в vzrosly-хуках) | `fonts.ts` |

**Добавить шрифт:** положить `.ttf` сюда → `load('Family','File.ttf')` в `fonts.ts` → экспорт константы → использовать в шаблоне. Лицензия — только OFL/свободная (репо публичный). Без неона — это про цвет, не про шрифт, но кегль/контраст держим читаемыми.
