# ShapeWipe — переход-вытеснение

Диагональная полоса в акцентном цвете проходит через кадр: к середине длительности
полностью перекрывает кадр, затем уходит, открывая. Задумывался как резкий переход (~0.9с).

## Статус: НЕ ГОДЕН ДЛЯ ПРОДА в текущем виде

Фидбэк yaromat (2026-06-14):

1. **Дёшево выглядит** — не дотягивает до уровня прода yaromat.
2. **Баг композита (общий для всех оверлеев):** хук НЕ играет ПОВЕРХ базового арта.
   Сейчас в момент `[at..at+dur]` кадр сменяется на чёрный фон с хуком, потом обратно —
   альфа ProRes 4444 теряется на шаге ffmpeg `overlay` в `tsx_overlay_job.py` (стр. ~109-110),
   overlay-поток приходит непрозрачным и кроет base. Должно: хук на фоне живого арта.
3. **Тест проводился на невалидной базе** — «затемнение» было виньеткой (центр остался
   светлым), светлый акцент терялся. Нужна равномерно тёмная база ИЛИ свежий тёмный арт.

## Что нужно для апрува
Починить альфа-композит (хук поверх арта), поднять уровень эффекта до прода
(вероятный путь — обучение на референс-клипах с Pinterest), валидный тест на тёмной базе.
Тогда `approved: yes` + повторное превью в треде 634.
