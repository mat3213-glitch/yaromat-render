# KineticCard — референс-шаблон

Полнокадровый шаблон-карточка (арт + заголовок трека + бренд). Написан мной как
ЭТАЛОН контракта для mimo (граница песочницы, демонстрация props/rng/format).

## Статус: ТОЛЬКО РЕФЕРЕНС, не для прода

Не одобрен для боевого использования — служит образцом контракта `src/contract.ts`.
Для прод-типографики используется одобренный [KineticWords](KineticWords.md).
